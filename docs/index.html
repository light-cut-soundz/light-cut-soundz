Trim précis

Découpez votre audio par timestamp en secondes. Opération non-destructive sur le buffer en mémoire.

🔉

Fade in / Fade out

Rampe linéaire sur les N premières ou dernières secondes. Durée configurable au dixième de seconde.

🔊

Normalisation

Normalisation au pic max : tous les samples sont divisés par la valeur absolue maximale pour atteindre 0 dBFS.

⏩

Changement de vitesse

Accélération ou ralentissement par rééchantillonnage (rubato). 0.5× = moitié de durée, pitch monte aussi.

🎛️

Filtres DSP

Filtres biquad IIR : passe-bas, passe-haut, passe-bande. Basé sur le RBJ Audio Cookbook, Q = 0.707.

👂

Aperçu en temps réel

Écoutez l'effet des filtres, fades, vitesse et normalisation pendant la lecture via Web Audio API.

💾

Export multi-formats

Exportez en WAV (natif), MP3, FLAC ou OGG. WAV est disponible partout, les autres via FFmpeg si installé.

🦀

Décodage pur Rust

Lecture audio via symphonia — aucun GStreamer ni codec système requis. Fonctionne sur toutes les distros Linux.
